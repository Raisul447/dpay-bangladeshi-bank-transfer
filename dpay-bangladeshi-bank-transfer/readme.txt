A simple and reliable WooCommerce payment gateway plugin that enables Bangladeshi customers to pay via manual bank transfers using the DPay system.

== Description ==

**DPay Bangladeshi Bank Transfer** is a WooCommerce payment gateway plugin tailored for Bangladeshi eCommerce businesses. This plugin allows you to accept payments manually through direct bank transfers, following Bangladesh’s traditional banking system. It’s ideal for businesses that prefer receiving payments directly to their local bank accounts.

With DPay, customers will be provided with your bank account details at checkout. They can then complete the transfer manually through their preferred banking channel NPSB, RTGS, BEFTN.

== Features ==

- Seamlessly integrates with WooCommerce
- Accepts payments via manual bank transfers
- Supports multiple Bangladeshi banks
- Easy configuration and setup
- No third-party API required
- Fully compatible with most WordPress themes

== Supported Banks ==

You can input and display account information from any Bangladeshi bank. Common examples include:

- BRAC Bank Limited  
- Dutch-Bangla Bank Limited  
- Islami Bank Bangladesh Limited  
- Bank Asia Limited  
- Eastern Bank Limited  
- City Bank Limited  
- Standard Chartered Bangladesh  

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Navigate to **WooCommerce > Settings > Payments**.
4. Enable "**DPay Bangladeshi Bank Transfer**".
5. Enter your bank account details for customers to use during checkout.

== Frequently Asked Questions ==

= Will this plugin work with any WordPress theme? =  
Yes. It is compatible with any theme that supports WooCommerce.

= Is this plugin secure? =  
Yes. DPay follows WooCommerce’s standard practices for secure, manual payment gateways. Since no API or third-party service is involved, the transaction data remains within your control.
